EURO EXCHANGE ANALYSIS

About

This project contains Euro exchange data of various countries between 1999 and 2023 which can be used to analyze trends in any currency over time and apply machine learning models to predict future variations. The main objective of this dataset is to clean it first, then do some basic analysis, and finally apply ML models. The original dataset was sourced from Kaggle (click here).
